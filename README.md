## Usage: cpmk <language> <project_name>
C Example
```sh
cpmk c c_proj
```
C++ Example
```sh
cpmk cpp c_proj
```
or
```sh
cpmk cc c_proj
```

## Build it from source
### Requirements:
GNU Assembler
Linux probably ;)

```sh
git clone https://github.com/Sampie159/cpmkasm.git
cd cpmkasm
make install
```

## Uninstall
From the project directory
```sh
make uninstall
```

From anywhere in the system
```sh
sudo rm /usr/local/bin/cpmk
```
